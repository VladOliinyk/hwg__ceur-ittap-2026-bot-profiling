## 🏷️ Теги

### Типи задач
- `#feature` - Нова функціональність
- `#bug` - Виправлення багів
- `#improvement` - Покращення існуючого
- `#refactoring` - Рефакторинг коду
- `#documentation` - Документація
- `#testing` - Тестування

### Компоненти
- `#frontend` - Frontend частина
- `#backend` - Backend частина
- `#ui` - Користувацький інтерфейс
- `#api` - API
- `#database` - База даних

### Пріоритети
- `#critical` - Критичний
- `#high` - Високий
- `#medium` - Середній
- `#low` - Низький
